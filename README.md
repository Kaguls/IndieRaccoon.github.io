## 만들어 보고 싶은 게임



**첫 번째,** IP를 활용한 디펜스게임 

- ex)냥코대전쟁(The Battle Cats)

**두 번째**, 오픈월드 언리얼 게임 

- 젤다(Zelda), 팰월드(palword)

**세 번째**, 스토리 RPG게임

- ex)제노블레이드(Xenoblade)

**네 번째,** 생활 및 시뮬레이션 게임 

- ex) 마이 타임 앳 샌드록(My Time at Sandrock), 어게인 더 스톰(Again the storm)

**다섯 번째,** 힐링 농사게임 

- ex) 스타듀밸리(Stardew valley), 트래블러 레스트(Travellers Rest)

**여섯 번째,** MMORPG게임		

- ex)로스트아크(Lostark)
